# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

#### Text

- Very dark desaturated blue: hsl(238, 29%, 16%)
- Soft red: #f47c57

#### Gradient

Background gradient:

- Soft violet: #af67e9
- Soft blue: #6565e7

### Neutral

#### Text

- Very dark grayish blue: #4a4b5e

#### Dividers

- Light grayish blue: #e7e7e9
